### Cadastro / Atualizar:

> Pode ser relizado o cadastro e atualização de usuário no sistema.
>
>
>  #### Exemplos:
>  - Cadastrar:
>
> ```JSON
>{
>	"name": "FULANO de TAL",
>	"email": "fulanodeTAL@gmail.com",
>	"password": "123456"
>}
> ```
> - Atualizar:
> ``` JSON
> {
>   "name": "Fulano da Silva",
>	"email": "fulanoDaSilva@gmail.com",
>	"oldPassword": "123456",
>	"password": "123456",
>	"confirmPassword": "123456",
>	"avatar_id": 1
>}
> ```
